Frequently Asked Questions (FAQs)

1. What is Odoo + WhatsApp API integration?

This integration connects your Odoo ERP with WhatsApp Business API via Go4Whatsup. It allows automated communication such as sending quotations, invoices, delivery updates, payment reminders, and notifications directly from Odoo to your customers.

2. How does this integration help my business?

It automates repetitive messaging tasks, improves response times, reduces manual effort, and centralizes customer communication. Your team can track all messages, approvals, and document status directly in Odoo, boosting efficiency and customer satisfaction.

3. Do I need technical knowledge to set this up?

No technical expertise is needed. Our team handles the entire setup, including WhatsApp API activation, template configuration, trigger setup, and testing within Odoo.

4. Can I send bulk messages?

Yes. Approved WhatsApp templates can be used for bulk messages like promotions, notifications, or updates. You can also filter recipients by customer, vendor, or sales order criteria.

5. Can I preview WhatsApp messages before sending?

Yes. go4whatsup provides a live preview of each template, including text, images, and document attachments. You can edit or approve templates before they are sent automatically.

6. Can I receive customer replies in Odoo?

Absolutely. Incoming messages from customers are synced to Odoo’s activity log, enabling your team to respond, track conversations, and link replies to the corresponding order, invoice, or delivery.

7. How do trigger points work?

Trigger points allow you to set rules for sending messages automatically based on specific events such as Sales Order confirmation, Invoice validation, Payment receipt, Delivery status updates, POS bills, and RFQs. Each trigger can have a different template and media attached.

8. Does it work with older Odoo versions?

Yes. The module is compatible with Odoo versions 15, 16, 17, 18, and 19.

9. How much does it cost?

The subscription is $29.99/month, including full setup, WhatsApp API configuration, template setup, and ongoing support.

10. Is this integration secure?

Yes. The integration uses the official WhatsApp Business API via Go4Whatsup. All messages and customer data are encrypted, fully compliant with WhatsApp policies, and tracked within Odoo.

11. Can I send media files along with messages?

Yes. You can attach PDFs, images, or documents to messages. Each media file can be linked to a specific template or trigger point.

12. What happens if a message fails to send?

Failed messages are logged in the activity log with details, allowing your team to resend or troubleshoot the issue directly from Odoo.
