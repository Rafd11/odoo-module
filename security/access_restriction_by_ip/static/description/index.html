Restrict User Access by IP Address

This module allows administrators to restrict user access based on IP addresses. You can assign multiple IP addresses for each user.

Key Features:
  • Restrict login to specific IPs
  • Supports multiple IPs
  • Automatic logout on unauthorized IP
  • Enhances system security

The module integrates with existing user management and enforces restrictions in real-time.
